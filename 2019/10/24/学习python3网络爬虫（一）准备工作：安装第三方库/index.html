学习python3网络爬虫（一）准备工作：安装第三方库

发表于 2019-10-24 更新于 2019-10-25 分类于 爬虫 阅读次数： Valine： 本文字数： 1.9k

python3网络爬虫中所需的第三方相关库

  • 请求库安装（requests、selenium、aiohttp）
  • 浏览器插件安装（ChromeDriver）
  • 解析库安装（lxml、Beautifule Soup、pyquery、tesserocr）
  • 数据库安装
  • 存储库安装
  • WEB库安装

爬虫可以简单分为几步：抓取页面、分析页面和储存数据。

在抓取页面的过程中，我们需要模拟浏览器向服务器发出请求，所以需要用到一些python库来实现HTTP请求操作。安装所需的一些第三方库如下：

请求库安装（requests、selenium、aiohttp）

请求库 pip安装 功能
requests pip3 install requests 用于http请求的模块
selenium pip3 install selenium 自动化测试工具，驱动浏览器做如点击、下拉等操作
aiohttp pip3 install aiohttp 异步web服务库，提高页面数据抓取效率

解析库安装（lxml、Beautifule Soup、pyquery、tesserocr）

解析库 pip安装 功能
lxml pip3 install lxml HTML和XML解析，支持XPath解析方式，解析效率较高
Beautifule Soup pip3 install beautifulsoup4 HTML或XML解析库。强大API和多样解析方式
pyquery pip3 install pyquery jQuery类似解析HTML文档，支持CSS选择器

tesserocr是Python的一个OCR识别库，识别验证码转化为文字，但其实是对tesseract做的一层Python API封装，所以他的核心是tesseract。因此，在安装tesserocr之前，需要安装tesseract：

tesseract官方下载地址

(dev为开发版，选不是dev版本的，我这里选择比较新的64位，4.0版本：tesseract-ocr-w64-setup-v4.0.0.20181030.exe)

tesseract安装成功后再安装tesserocr，windows下安装tesserocr使用pip的方式会各种报错，为方便所以此处采用whl方式安装。
